# Python PDF Invoice Automation

This project demonstrates how Python can automatically extract structured data from invoice PDF files.

## What the script does
- Reads invoice PDFs
- Extracts key fields (invoice number, date, total amount)
- Converts data into CSV format
- Automates document processing

## Technologies
- Python
- PyPDF2
- pandas

## Use cases
- Invoice processing
- Accounting automation
- Admin workflows
- Document data extraction

## How to run
```bash
pip install PyPDF2 pandas
python extract_pdf.py
